# Valley of Tea Dragons

Среди всех известных мне регионов Долина Чайных Драконов остаётся одним из наиболее изолированных и древних мест, упоминаемых в сохранившихся записях. Именно здесь находится Первое Чайное Дерево, вокруг которого спустя многие поколения сформировались Деревня Селян, чайные поля и большая часть современной жизни Долины.

Большая часть истории региона до сих пор остаётся фрагментарной. Многие легенды противоречат друг другу, старые тропы давно утрачены, а происхождение самой Долины продолжает вызывать множество вопросов даже среди местных жителей.
