# Home's Journey Archive

A lore archive for the Home's Journey universe, built with Jekyll + Just the Docs.

🌐 [Visit the Archive](https://homejourneybook.github.io/Homes-journey-archive/)

## Structure
- `Artifacts/` - records about Artifacts
- `Concepts/` — key concepts and terms
- `Tavern/` — records about the Tavern and Home Travelers
- `Valley/` - sub records from Worlds, aboth the Valley
- `Worlds/` — records about known Home Worlds
- `_includes` - setting files
- `_layouts` - setting files
- `assets` - setting files
