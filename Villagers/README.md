# Селяне

Селяне являются основными жителями Долины Чайных Драконов. На протяжении многих поколений именно они поддерживают существование Деревни, ухаживают за Чайными Полями, сохраняют старые традиции и занимаются ремёслами, связанными с жизнью вокруг Первого Чайного Дерева.

Большинство Селян носит плотные одежды и маски, форма которых напрямую связана с их ремеслом или обязанностями в пределах Долины. Несмотря на различия профессий, сами жители редко разделяют друг друга по статусу, воспринимая жизнь Деревни как единый общий порядок.

Значительная часть знаний Селян никогда не записывалась напрямую. Многие традиции, церемонии и ремёсла продолжают существовать исключительно через практику, совместную работу и устные рассказы, постепенно передаваемые между поколениями.

На данный момент происхождение ряда обычаев и символов Долины остаётся неизвестным даже для самих жителей.
